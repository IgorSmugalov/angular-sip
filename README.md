# AngularSip

Клиент для тестирования Sip в связке с Angular, JsSIP и RxJS

[Онлайн-версия](https://igorsmugalov.github.io/angular-sip/)

## Запуск локально

1. #### Установка зависимостей
```bash
npm install
```
2. #### Запуск дев-сервера

```bash
npm start
```
3. #### [http://localhost:4200/](http://localhost:4200/)
